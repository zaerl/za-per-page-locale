== Description ==

### Per Page Locale ###

This free plugin offers a simple way to change the site locale per page.

== Installation ==

1. To begin, click on the Plugins link in the left hand sidebar, then click Add New.
2. Search for "Per Page Locale". The latest version will be in the search results. Click the "Install Now" button.
3. Next, click the "Activate" button.
